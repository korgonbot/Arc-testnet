Production-ready checklist

These upgrades move the launch from prototype to a safer mainnet-ready flow.

  • Merkle GTD whitelist — replace the placeholder GTD logic with verified proofs.
  • Per-wallet GTD allocation — cap each allowlisted wallet to a fixed minted quantity.
  • Team reserve cap — reserve a defined share of the supply for team or future drops.
  • USDC decimals validation — ensure pricing and transfers match the exact token standard.
  • Batch admin tools — support airdrops, metadata updates, and bulk operations.
  • Hardhat tests — cover success paths, revert cases, and security edge cases.
